# 🌈 南京天气预报 - 小朋友版

这是一个专为 6 岁以内小朋友设计的卡通风格天气预报应用！

## 🎨 特色功能

- **卡通风格界面** - 色彩鲜艳，可爱有趣
- **大图标显示** - 小朋友容易识别
- **动画效果** - 漂浮的云朵、闪烁的星星
- **简单易懂** - 用 emoji 和简单文字展示天气
- **默认南京** - 自动显示南京天气
- **未来预报** - 显示今天和未来 4 天的天气

## 🚀 如何使用

### 方法 1：直接打开
双击 `index.html` 文件，用浏览器打开即可！

### 方法 2：在浏览器中输入
在浏览器地址栏输入：
```
file:///C:/Users/Administrator/.openclaw/workspace/weather-app/index.html
```

## 📱 支持的设备

- 💻 电脑（Windows/Mac）
- 📱 平板（iPad/Android）
- 📲 手机（需要调整屏幕大小）

## 🌤️ 显示内容

- 当前温度
- 天气状况（用可爱的 emoji 表示）
- 湿度
- 风速
- 最高/最低温度
- 未来 4 天预报

## 🎯 设计说明

专门为小朋友设计：
- 使用圆角和柔和的色彩
- 大字体，容易阅读
- 有趣的动画吸引注意力
- 简单的信息展示，不会太复杂

## 🔄 刷新天气

点击底部的"🔄 刷新天气"按钮可以获取最新天气数据！
